open

Function of Titanium.UI.Tab.
PlatformSince
Android0.8
iPhone0.8
iPad0.8
Mobile Web1.8

Summary

Opens a new window.


Description

On iOS and Mobile Web, the new window is opened as the top window in the tab's window stack. On Android, the new window is opened as a new, heavyweight window, obscuring the tab group.


Arguments

NameTypeSummary
windowTitanium.UI.Window

Window to open.

optionsObject

Dictionary of display and animation settings to use when opening the window. Identical to the options parameter to Window.open.


Return Type

void

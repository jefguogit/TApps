openMusicLibrary

Function of Titanium.Media.
PlatformSince
iPhone0.1
iPad0.1

Summary

Shows the music library and allows the user to select one or more tracks.


Description

If autohide is set to false, you must hide the library explicitly using hideMusicLibrary.


Arguments

NameTypeSummary

touchmove

Event of Titanium.UI.PickerColumn.
PlatformSince
Android0.9
iPhone0.9
iPad0.9
Mobile Web1.8

Summary

Fired as soon as the device detects movement of a touch.


Description

Event coordinates are always relative to the view in which the initial touch occurred


Properties

NameTypeSummary
globalPoint Deprecated

Dictionary with properties x and y describing the location of the event in screen coordinates.

sourceObject

Source object that fired the event.

typeString

Name of the event fired.

xNumber

X coordinate of the event from the source view's coordinate system.

yNumber

Y coordinate of the event from the source view's coordinate system.

load

Event of Titanium.UI.WebView.
PlatformSince
Android0.8
iPhone0.8
iPad0.8
Mobile Web1.8

Summary

Fired when the web view content is loaded.


Description

For Mobile Web, the url returned in the load event is generally accurate when loading same domain pages and remote pages that you explicitly set using the url property. If you navigate to a remote page, then this url is set to the initial remote page's URL.


Properties

NameTypeSummary

Source object that fired the event.

Name of the event fired.

URL of the web document.

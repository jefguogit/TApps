createTCPSocket

Function of Titanium.Network.
PlatformSince
iPhone0.1
iPad0.1
Mobile Web1.8

Summary

Deprecated

Use Titanium.Network.Socket.createTCP instead.


Description

Legacy method to create and return an instance of Titanium.Network.TCPSocket.

Note that TCPSocket is still used with the Bonjour services. For other uses, see Titanium.Network.Socket.TCP.


Arguments

NameTypeSummary
hostNameString

Host name to connect to/listen on.

portNumber

Port for the socket.

modeNumber

Socket's mode; one of Titanium.Network.READ_MODE, Titanium.Network.WRITE_MODE, Titanium.Network.READ_WRITE_MODE.

parametersDictionary<Titanium.Network.TCPSocket>

Dictionary object with properties defined in Titanium.Network.TCPSocket.


Return Type

Titanium.Network.TCPSocket

format

Function of Global.String.
PlatformSince
Android0.8
iPhone0.8
iPad0.8

Summary

Formats a string using printf-style substitution.


Description

The format string follows the IEEE printf specification.

For each "conversion specification" (ie. %s for a string or %d for a number) used inside the string, formatString argument, the respective value is substituted from the second argument list. For example:

var forename = 'Paul';
var number = 21;
var message = String.format('Welcome, %s! You are visitor number %d', forename, number);
Ti.API.info(message);

Arguments

NameTypeSummary
formatStringString

An IEEE printf-style string, containing zero or more conversion specifications.

valueString Number

Comma-separated list of one or more values to substitute into the formatString. Optional on Android.


Return Type

String

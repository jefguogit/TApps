add

Function of Titanium.UI.TextArea.
PlatformSince
iPhone0.8
iPad0.8

Summary

Adds a child to this view's hierarchy.


Description

Adding children to a TextArea is not supported on all platforms.

If you need to display views on top of this object, consider using another view as a container for both this object and the views you want to appear on top of it.


Arguments

NameTypeSummary
viewTitanium.UI.View

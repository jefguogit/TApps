Titanium.UI.TabbedBar

object

Summary

Deprecated

Use Titanium.UI.iOS.TabbedBar instead.


Description

This object has been deprecated and moved to Titanium.UI.iOS.TabbedBar.

A tabbed bar is a button bar that maintains a state (visually distinguished as a pressed or selected look). It is closely related to the ButtonBar control. See the description of ButtonBar for information on styling tabbed bars and buttons bars.

Use the Titanium.UI.createTabbedBar method to create a Tabbed Bar.


Code Examples

Simple Tabbed Bar with 3 items

var bb1 = Titanium.UI.createTabbedBar({ labels:['One', 'Two', 'Three'], backgroundColor:'#336699', top:50, style:Titanium.UI.iPhone.SystemButtonStyle.BAR, height:25, width:200 }); win.add(bb1);

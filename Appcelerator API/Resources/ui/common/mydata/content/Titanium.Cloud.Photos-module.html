Titanium.Cloud.Photos

module

Summary

Provides methods for accessing ACS photos.


Code Examples

Create Photo

This example creates a new photo and checks the response.

Cloud.Photos.create({
    photo: Titanium.Filesystem.getFile('photo.jpg')
}, function (e) {
    if (e.success) {
        var photo = e.photos[0];
        alert('Success:\\\
' +
            'id: ' + photo.id + '\\\
' +
            'filename: ' + photo.filename + '\\\
' +
            'size: ' + photo.size,
            'updated_at: ' + photo.updated_at);
    } else {
        alert('Error:\\\
' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

Search for Photos

This example searches for photos and checks the response.

Cloud.Photos.search({
    user_id: savedUserId
}, function (e) {
    if (e.success) {
        alert('Success:\\\
' +
            'Count: ' + e.photos.length);

Query for Photos

This example requests a list of photos and checks the response.

    page: 1,
    per_page: 20,

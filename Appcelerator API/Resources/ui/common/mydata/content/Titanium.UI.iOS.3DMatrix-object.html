Titanium.UI.iOS.3DMatrix

object

Summary

Deprecated

Use Titanium.UI.3DMatrix instead.


Description

The 3D Matrix is an object for holding values for a 3D affine transform.

The 3DMatrix is created by Titanium.UI.iOS.create3DMatrix. A 3D transform is used to rotate, scale, translate, or skew the objects in three-dimensional space. A 3D transform is represented by a 4 by 4 matrix.

You create an identity matrix by creating a 3D Matrix with an empty constructor.

See Titanium.UI.3DMatrix for sample usage.

Titanium.Gesture

module

Summary

The Gesture module is responsible for high-level device gestures such as orientation changes and shake gestures.


Properties

NameTypeSummary
landscapeBoolean

Indicates whether current window is considered landscape by the device.

read-only
orientationNumber

Orientation of the current window.

read-only
portraitBoolean

Indicates whether current window is considered portrait by the device.

read-only

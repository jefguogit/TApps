Titanium.UI.MaskedImage

object

Summary

A control that displays an image composited with a background image or color.


Description

This is an iOS-specific control that can be used to display an image combined with another image and/or color. The layers are drawn in the following order:

  • The mask, or background image.
  • The image, or foreground image.
  • The tint, or constant tint color.

The way the layers are combined depends on the value of the mode property. The mode value can be set to one of the Titanium.UI.iOS BLEND_MODE constants. These constants correspond directly to the iOS blend modes described in the iOS developer library:

"CGBlendMode" in CGContext Reference

and:

"Using Blend Modes with Images" in Quartz2D Reference

When compositing two images, the mask property specifies the background, or destination (D) image, and the image property specifies the foreground, or source (S) layer. For example, to use an image as an alpha mask for another image, you could use the following code:

var imageMask = Titanium.UI.createMaskedImage({
    mask : 'mask.png', // alpha mask
    image : 'demo_image.png', // image to mask
    mode : Titanium.UI.iOS.BLEND_MODE_SOURCE_OUT
});

Note that this legacy control lacks many of the features associated with a standard ImageView control.

Use the Titanium.UI.createMaskedImage method to create a masked image view.


Code Examples

Luminosity Blend Mode

The following code excerpt creates a monochrome version of the background image, using the luminosity from the background image and the specified tint color:

var imageMask = Titanium.UI.createMaskedImage({
    mask : 'demo_image.png', // background image
    tint: 'red',
    mode : Titanium.UI.iOS.BLEND_MODE_LUMINOSITY,
});

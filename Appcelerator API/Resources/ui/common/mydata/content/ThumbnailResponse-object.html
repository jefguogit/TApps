ThumbnailResponse

object

Summary

Simple object passed to the thumbnail callback in response to therequestThumbnailImagesAtTimesmethod.


Description

The callback is invoked when a movie thumbnail is available.


Properties

NameTypeSummary
errorString

If success is false, provides an error message.

Thumbnail image, as a Blob.

successBoolean

Boolean to indicate success or failure.

timeNumber

Time offset for the thumbnail, in seconds.

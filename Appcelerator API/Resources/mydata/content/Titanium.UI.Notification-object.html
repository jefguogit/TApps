Titanium.UI.Notification

object

Summary

An Android Toast-style notification.


Description

A Toast notification is an unobtrusive, pop-up notification that doesn't block the UI. Use the Titanium.UI.createNotification method to create a Toast notification.

By default, a Toast notification appears centered on the bottom half of the screen.

This object will be moved to the Android namespace in a future release.


Code Examples

Simple Toast Notification

The simplest use case is also the most common.

var toast = Ti.UI.createNotification({
    message:"Please Stand By",
    duration: Ti.UI.NOTIFICATION_DURATION_LONG
});
toast.show();

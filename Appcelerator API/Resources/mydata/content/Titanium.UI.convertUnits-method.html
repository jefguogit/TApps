convertUnits

Function of Titanium.UI.
PlatformSince
Android2.0.0
iPhone2.0.0
iPad2.0.0
Mobile Web2.0.0

Summary

Converts one type of unit to another using the metrics of the main display.


Description

As this method does not support percentages, 0 is returned if they are specified.


Arguments

NameTypeSummary
convertFromValueString

Measurement and optional unit to convert from, i.e. 160, "120dip". Percentages are not accepted.

convertToUnitsString

Desired unit for the conversion result. Use one of the unit constants in Titanium.UI.


Return Type

Number

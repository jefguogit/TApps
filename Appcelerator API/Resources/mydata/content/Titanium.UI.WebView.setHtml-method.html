setHtml

Function of Titanium.UI.WebView.
PlatformSince
iPhone0.8
iPad0.8
Mobile Web1.8

Summary

Sets the value of html property.


Description

In iOS, there are additional parameters baseURL and mimeType that can be used to configure the WebView main content presentation. For example:

setHtml('<html><body>Hello, <a href="/documentation">Titanium</a>!</body></html>',
        {baseURL:'http://developer.appcelerator.com/'})

Arguments

NameTypeSummary
htmlObject

New HTML to display in the web view.

baseURLString

The base URL for the content. Only used by iOS.

mimeTypeString

The MIME type of the content. Only used by iOS.


Return Type

void

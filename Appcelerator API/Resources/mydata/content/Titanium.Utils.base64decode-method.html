base64decode

Function of Titanium.Utils.
PlatformSince
Android0.9
iPhone0.9
iPad0.9
Mobile Web1.8

Summary

Returns the specified data decoded from Base64.


Description

On Mobile Web, if obj is a [Titanium.Blob] containing binary data, then obj is returned unmodified because the binary blob would be decoded only to be re-encoded in the returned blob object.


Arguments

NameTypeSummary
objString Titanium.Blob

Data to decode.


Return Type

Titanium.Blob

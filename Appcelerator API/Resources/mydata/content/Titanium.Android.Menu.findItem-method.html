findItem

Function of Titanium.Android.Menu.
PlatformSince
Android1.5

Summary

Locates a MenuItem in this menu, by item ID or reference.


Description

Returns the identified menu item, or null if the menu item was not located.


Arguments

NameTypeSummary
itemNumber Titanium.Android.MenuItem

Integer itemId or a reference to a MenuItem object.


Return Type

Titanium.Android.MenuItem

Titanium.Cloud.KeyValues

module

Summary

Provides methods for accessing ACS Key-Values storage.


Code Examples

Set a Key-Value

This example sets a key-value and checks the response.

Cloud.KeyValues.set({
    name: 'welcome_message',
    value: 'Welcome to Appcelerator Cloud Services'
}, function (e) {
    if (e.success) {
        alert('Success');
    } else {
        alert('Error:\\\
' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

Get a Key-Value

This example gets a key-value and checks the response.

Cloud.KeyValues.get({
    name: 'welcome_message'
}, function (e) {
    if (e.success) {
        var keyvalue = e.keyvalues[0];
        alert('Success:\\\
' +
            'name: ' + keyvalue.name + '\\\
' +
            'value: ' + keyvalue.value);
    } else {
        alert('Error:\\\
' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

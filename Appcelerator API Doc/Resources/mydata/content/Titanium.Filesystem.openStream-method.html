openStream

Function of Titanium.Filesystem.
PlatformSince
Android0.1
iPhone0.1
iPad0.1
Mobile Web1.8

Summary

Opens file using the Titanium.IOStream interface.


Arguments

NameTypeSummary
modeNumber

Access mode. Either Titanium.Filesystem.MODE_READ, Titanium.Filesystem.MODE_WRITE, or Titanium.Filesystem.MODE_APPEND.

pathString

One or more path components. Path arguments are joined together using the platform-specific path separator to form the full path.


Return Type

Titanium.Filesystem.FileStream

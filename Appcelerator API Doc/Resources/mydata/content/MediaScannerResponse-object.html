MediaScannerResponse

object

Summary

Simple object passed to the scanMediaFiles callback.


Description

The callback is called once per scanned file.


Properties

NameTypeSummary
pathString

Path to the media file that was scanned.

uriString

URI to the file if it was scanned and added to the media library, or nullif the file was not added.

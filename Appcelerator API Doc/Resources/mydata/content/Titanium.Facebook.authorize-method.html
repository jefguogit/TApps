authorize

Function of Titanium.Facebook.
PlatformSince
Android0.8
iPhone0.8
iPad0.8

Summary

Prompts the user to log in (if not already logged in) and authorize your application.


Description

Be sure to set your appid and your required [permissions](Titanium.Facebook.permissions] before calling authorize.

A login event is generated to indicate successful or unsuccessful login attempt.


Return Type

void

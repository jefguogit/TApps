Titanium.XML.CDATASection

object

Summary

Used to include blocks of literal text containing characters that would otherwise needto be escaped.


Description

Implements the DOM Level 2 API on Android and iOS with some non-standard extensions. Exposes the DOM Level 3 API implementation on Mobile Web. On Mobile Web, there is no Titanium.XML.CDATASection object. A string object containing the data is used directly instead.

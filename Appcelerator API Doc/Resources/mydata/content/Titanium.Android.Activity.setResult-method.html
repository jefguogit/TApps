setResult

Function of Titanium.Android.Activity.
PlatformSince
Android1.5

Summary

Sets the result of this activity using an Intent.


Description

This should be used in the case when the Activity responds to startActivityForResult.

Also see Android's documentation for setResult.


Arguments

NameTypeSummary
resultCodeNumber

Result code for this activity. Can be Titanium.Android.RESULT_OK, Titanium.Android.RESULT_CANCELED, or Titanium.Android.RESULT_FIRST_USER.

intentTitanium.Android.Intent

An optional Intent with extra result data.


Return Type

void

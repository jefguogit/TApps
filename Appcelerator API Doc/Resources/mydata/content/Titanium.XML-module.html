Titanium.XML

module

Summary

The top level XML module. The XML module is used for parsing and processing XML-based content.


Description

The API for this module is based on the W3C DOM specification.

Android and iOS implement the DOM Level 2 specification with some non-standard extensions, which are documented in the appropriate places.

Both iOS and Android lack DTD support.

Mobile Web directly exposes the DOM Level 3 implementation in the browser. New features introduced in DOM Level 3 are not documented here for brevity. For information on these features, please use the W3C documentation.

setOnClickPendingIntent

Function of Titanium.Android.RemoteViews.
PlatformSince
Android1.6

Summary

Launches a Titanium.Android.PendingIntent when the specified view is clicked.


Description

See Android's documentation for setOnClickPendingIntent.


Arguments

NameTypeSummary
viewIdNumber

Resource ID of the view to add a click listener to.Application-specific resource IDs can be specified using Titanium.App.Android.R.

pendingIntentTitanium.Android.PendingIntent

The PendingIntent to execute when this view is clicked.


Return Type

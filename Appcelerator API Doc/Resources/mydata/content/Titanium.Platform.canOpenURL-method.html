canOpenURL

Function of Titanium.Platform.
PlatformSince
iPhone0.1
iPad0.1

Summary

Returns whether the system is configured with a default application to handle the URL's protocol.


Description

On Mobile Web, this method always returns true, even if the browser can't open the specific URL, unless the URL is empty or undefined.


Arguments

NameTypeSummary
urlString

The url to check.


Return Type

Boolean

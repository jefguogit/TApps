Titanium.Media.MusicPlayer

object

Summary

This object represents a music controller.


Description

A MusicPlayer lets you manage and playback a queue of media Item objects.

Use the Titanium.Media.createMusicPlayer method to create a music palyer.

Use the Media module's openMusicLibrary and queryMusicLibrary methods to access the media items in the device's media library.


Properties

NameTypeSummary
currentPlaybackTimeNumber

Current point in song playback, in seconds.

nowPlayingTitanium.Media.Item

An Item object representing the currently playing media item.

read-only
playbackStateNumber

Playback state: one ofMUSIC_PLAYER_STATE_STOPPED,MUSIC_PLAYER_STATE_PLAYING,MUSIC_PLAYER_STATE_PAUSED,MUSIC_PLAYER_STATE_INTERRUPTED,MUSIC_PLAYER_STATE_SEEK_FORWARD,orMUSIC_PLAYER_STATE_SEEK_BACKWARD.

read-only
repeatModeNumber

Repeat setting: one ofMUSIC_PLAYER_REPEAT_DEFAULT,MUSIC_PLAYER_REPEAT_NONE,MUSIC_PLAYER_REPEAT_ONE, orMUSIC_PLAYER_REPEAT_ALL.

shuffleModeNumber

Shuffle setting: one ofMUSIC_PLAYER_SHUFFLE_DEFAULT,MUSIC_PLAYER_SHUFFLE_NONE,MUSIC_PLAYER_SHUFFLE_SONGS, orMUSIC_PLAYER_SHUFFLE_ALBUMS.

volumeNumber

Volume level for the music player, from 0.0 (muted) to 1.0 (loudest).
